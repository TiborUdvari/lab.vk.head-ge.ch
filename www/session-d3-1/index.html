SVG

  • SVG from scratch sur MDN, référence des éléments, référence des attributs
  • Conteneurs :
    • svg : viewBox, preserveAspectRatio, width, height
    • g : transform (translate(x, y), rotate(deg), scale(k))
  • Formes simples :
    • rect : x, y, width, height, rx, ry
    • circle : cx, cy, r
    • ellipse : cx, cy, rx, ry
    • line : x1, y1, x2, y2
    • polygon / polyline : points
  • path :
    • attribut d, commandes M (move), L (line), H/V (lignes horizontales/verticales), C/S (cubique Bézier), Q/T (quadratique Bézier), A (arc), Z (close)
    • tutoriel paths sur MDN
  • Texte :
    • text : x, y (origine : la baseline par défaut), dx, dy (offsets, listes par caractère possibles), font-size, font-family
    • tspan : texte multiligne (x reset + dy="1.2em" par ligne)
    • Alignement : text-anchor (start / middle / end), dominant-baseline (auto / middle / hanging / text-before-edge)
  • Style :
    • Fill : fill, fill-opacity
    • Stroke : stroke, stroke-width, stroke-opacity, stroke-linecap (butt / round / square), stroke-linejoin (miter / round / bevel)
    • Pointillés : stroke-dasharray, stroke-dashoffset
    • Global : opacity
  • Définitions réutilisables :
    • defs : conteneur pour définitions réutilisables
    • clipPath + clip-path : limiter le dessin à une zone (utile pour clipper un graphique à sa zone interne)
    • linearGradient / radialGradient + stop : dégradés
    • pattern : motifs répétés en remplissage
    • marker : flèches/symboles aux extrémités d'un path
    • use : réutiliser un élément défini ailleurs

Exercice (en binômes) : « Oui, mais... »

Tout graphique avec un agenda laisse quelque chose de côté. Choisissez-en un qui essaie de vous convaincre dans n'importe quelle source qui porte un message, et reproduisez-le en D3 (pas pixel-perfect, juste les mêmes données ; les retrouver fait partie du travail). Créez ensuite une seconde visualisation qui propose une lecture plus complète du même sujet : variable cachée, échelle trompeuse, période, comparaison absente. Le passage d'une lecture à l'autre peut se faire par une transition animée (clic, scroll, boucle) ou par les deux graphiques en parallèle, selon ce qui sert mieux le propos. Contextualiser et analyser les choix. Cas d'étude d'actualité : « Pas de Suisse à 10 millions ».

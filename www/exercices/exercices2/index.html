Exercices JavaScript débutants – Chargement et manipulation de données JSON

Dans cette série d’exercices, nous allons apprendre à charger des données au format JSON et à les manipuler avec JavaScript. Vous devez compléter le fichier exercices.js, qui contient des fonctions utilisées dans les différentes pages.

L’objectif principal est de se concentrer sur JavaScript, mais il est recommandé de regarder le code HTML pour mieux comprendre comment structurer vos pages et afficher les résultats.

Nous allons utiliser des données de prénoms de filles fournies par la Confédération.

Pages d’exercices

  1. Page 1 – Chargement des données : récupérer et afficher les données JSON brutes.
  2. Page 2 – Extraire les noms (Map to Names) : travailler sur le tableau des prénoms et les trier.
  3. Page 3 – Filtrer par lettre initiale : filtrer les prénoms qui commencent par une lettre donnée.
  4. Page 4 – Vue objet composé

Chaque page correspond à une étape de l’exercice. Complétez les fonctions dans exercices.js et vérifiez vos résultats en ouvrant la page correspondante.

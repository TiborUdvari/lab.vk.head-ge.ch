Partie 3 – Web scraping

Exercice (en binômes)

  • Créer ou augmenter un dataset (collecte, scraping ou LLM).
  • Utiliser ce dataset pour produire une page web interactive, responsive et publiable, avec une visualisation de données.
  • Documenter les difficultés liées à la collecte ou à la structuration du dataset.
  • Réfléchir : comment ces biais ou contraintes peuvent-ils se retrouver dans d’autres datasets ?

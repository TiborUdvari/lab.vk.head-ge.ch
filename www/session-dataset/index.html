Exemple : Best-Selling Video Games

Pour la prochaine fois :

  • Choisissez un dataset simple téléchargeable sur Wikipédia. Exemples : Oscars remportés, films les plus rentables, chansons les plus longues, etc.
  • Nettoyez et préparez votre dataset (supprimez les doublons, uniformisez les catégories, vérifiez les valeurs manquantes, reformatez si nécessaire).
  • Importez le dataset dans Observable, réalisez au moins deux visualisations de données, commentez l’origine des données et contextualisez-les.
  • Intégrez vos visualisations et vos observations dans une page web responsive, enrichie de contenus médias pour renforcer vos arguments.
